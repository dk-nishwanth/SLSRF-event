# SLSRF Research Foundation Website

A professional and modern website for Semporutkalai Life Science Research Foundation (SLSRF), featuring a corporate aesthetic, responsive layout, and interactive components.

## Features

- Modern, responsive design with smooth animations
- Event gallery and speaker sections
- Contact forms and location map integration
- WhatsApp integration for quick communication

## Run Locally

**Prerequisites:** Node.js (v18 or higher)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deploy to Vercel

### Option 1: Deploy via Vercel Dashboard

1. Push your code to GitHub
2. Go to [Vercel](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repository
5. Click "Deploy"

### Option 2: Deploy via Vercel CLI

1. Install Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Login to Vercel:
   ```bash
   vercel login
   ```

3. Deploy:
   ```bash
   vercel
   ```

## Build for Production

```bash
npm run build
```

The build output will be in the `dist` directory.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS (via inline styles)
- Lucide React Icons
